
# Genomic Prediction and Selection Manual

Welcome! This book will teach an entire genomic prediction and selection workflow. Target audience are students of plant breeding and quantitative genetics as well as biostatisticians/informaticians tasked as responsible for genomic evaluations in their breeding program.

The pipeline this book will center on is as executed for the [NextGen Cassava Breeding Project](https://www.nextgencassava.org/) partner programs.
